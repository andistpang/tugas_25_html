Mengenal Semantic Elements di HTML 5

  • Article

    • Ketika kita ingin membuat layout untuk menampilkan block artikel-artikel yang ada di website kita, tag artcile bisa kita gunakan dengan element header dan footer. Karena tag article memiliki type block, artinya elemen ini memiliki lebar sesuai lebar brwoser jika kita tidak menentukan lebarnya.

  • Aside

    • Element ini bisa kita gunakan untuk melengkapi layout utama. Misal kita memiliki halaman blogmaka kita bisa menambahkan aside dibagian bawah postingan untuk menjelaskan identitas penulis artikel. Karena aside memiliki type/sifat block maka lebarnya sesuai page jika kita tidak menentukan lebarnya.
